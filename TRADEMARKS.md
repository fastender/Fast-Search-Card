## What this means in practice

The **code** of this project is licensed under [GPL-3.0-or-later](LICENSE). You are free to use, modify, and redistribute the code under the terms of that license.

The **name and branding** are not part of that license. Forks and derivative works:

- **may not** use the name "Fast Search Card" or any confusingly similar name (e.g. "FastSearch Card", "Fast Search HA", "Fast Search Pro").
- **may not** use the project logo or marketing screenshots without permission.
- **may not** present themselves in a way that suggests endorsement by or affiliation with the original project.
- **must** use a clearly distinct name and branding when distributed publicly (for example via HACS).

You **may** mention the original project for context — e.g. "based on Fast Search Card by fastender" — as long as it's clear that your fork is a separate work.

## Why

The project author has invested years of work in building a recognizable identity for Fast Search Card. The trademark notice exists so that users can trust that anything carrying the "Fast Search Card" name comes from — or is endorsed by — the original author.

This is the same model used by major open-source projects: Mozilla Firefox, Docker, Kubernetes, and many others release their code under open-source licenses while protecting their names and logos as trademarks.

## Reporting violations

If you encounter a fork or derivative that violates this trademark notice — for example a HACS package using the "Fast Search Card" name without permission — please open an issue on the original repository or contact the project author.

For clear cases of trademark infringement combined with license violations (e.g. missing attribution, removed LICENSE file), a DMCA notice can be filed with GitHub at <https://github.com/contact/dmca>.
